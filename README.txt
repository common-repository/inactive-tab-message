Add custom message to browser tab if user click away to another tab

== Description ==

Ever wanted your site tab in browser to show some custom message when user click away to another tab ? Well now
you can with this super simple plugin. Just set custom message and you are ready to go.

== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress
2. Set custom message in 'Tools' submenu called Inactive Tab Messages
3. That's it! Enjoy custom tab message on frontend part of your site.

== Frequently Asked Questions ==

= How to set my custom message ?

Set custom message in 'Tools' submenu called Inactive Tab Messages

= Why this plugin don't show custom message when I'm looking at backend of site ? =

It's by design, it will show custom message only to frontend facing users.
